sublimetext-css3-snippets
=========================

Useful CSS3 Snippets for Sublime Text 2

## Installation

```
cd ~/Library/Application\ Support/Sublime\ Text\ 2/Packages/User/
git clone https://github.com/danielpquinn/sublimetext-css3-snippets.git
```

## Tab Triggers

__Border Radius__
```
border-radius
```
